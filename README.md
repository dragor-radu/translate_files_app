# Proiect Traducator de fisiere


## Instalare
1. Clonare:
 ```bash
    git clone https://github.com/dragor-radu/translate_files_app.git
    cd proiect-python
   ```

2. Instalare dependente:
 ```bash
    pip install -r requirements.txt
   ```

3. Exportare API:
* Windows: 
```set API_KEY=<API_KEY>```
* Linux:
```export API_KEY=<API_KEY>```

## Rulare:
```bash
python3 main.py
```
